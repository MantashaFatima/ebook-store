# StudyBooks - E-Book Marketplace

StudyBooks is a responsive E-Book Marketplace web application built using React.js and Tailwind CSS. The platform allows users to discover, explore, and purchase educational e-books for competitive exam preparation.

## Live Demo

https://ebook-store-app.netlify.app/

---

## Features

### Home Page
- Hero Banner
- Search Books
- Popular Categories
- Featured E-Books
- Best Selling Books
- Call To Action Section

### E-Book Listing Page
- Search Books
- Sort By Price (Low to High / High to Low)
- Grid / List View Toggle
- Pagination
- Responsive Layout

### Book Detail Page
- Book Cover & Information
- Author Details
- Book Preview Section
- Ratings & Reviews
- Related Books
- Buy Now Option

### Checkout Page
- Billing Information Form
- Coupon Code Section
- Payment Method Selection
- Order Summary
- Purchase Confirmation Flow

### My Library
- Purchased Books Collection
- Search Library
- Continue Reading
- Download Books

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd ebook-store
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

The application will run on:

```bash
http://localhost:3000
```

---

## Production Build

```bash
npm run build
```

The optimized production build will be generated inside the `build` folder.
